# RubyCqrs

A Ruby implementation of CQRS, using event sourcing. go wiki cqrs for more information.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruby_cqrs'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruby_cqrs
